# Meu Portfólio

  Aplicação de portfólio pessoal desenvolvido com React, TypeScript e CSS Modules. O projeto foi criado com o objetivo de apresentar minhas informações profissionais, habilidades técnicas, projetos desenvolvidos e formas de contato em uma página organizada e responsiva.

## Sobre o projeto

O portfólio conta com seções bem definidas, incluindo apresentação inicial, sobre mim, projetos, habilidades, contatos e rodapé. Cada seção foi construída em componentes separados, facilitando a manutenção e a organização do código.

A seção de projetos exibe cards com imagens, descrições e links para os repositórios no GitHub, permitindo que visitantes conheçam melhor meus trabalhos e acessem o código-fonte de cada aplicação.

## Tecnologias utilizadas

- React
- TypeScript
- CSS Modules
- Vite
- HTML
- CSS
- JavaScript

## Funcionalidades

- Página inicial com apresentação pessoal
- Seção de projetos em formato de cards
- Links para repositórios no GitHub
- Seção de habilidades técnicas
- Seção de contatos
- Layout organizado com componentes reutilizáveis
- Estilização com CSS Modules

## Objetivo

O objetivo deste projeto é reunir meus principais trabalhos em um único lugar, demonstrando minha evolução como desenvolvedor front-end e servindo como uma apresentação profissional para oportunidades na área de tecnologia.

## Status do projeto

Projeto em desenvolvimento, com melhorias contínuas no layout, responsividade, organização do código e apresentação visual.
